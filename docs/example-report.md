# Example Release-Gate Report

This report is generated by the included sklearn example:

```bash
python examples/sklearn_example.py
```

The example trains a small RandomForest on synthetic data, records baseline performance,
then shifts several features to simulate drift before running the release gate. It is a
demo artifact, not a claim about a production model.
